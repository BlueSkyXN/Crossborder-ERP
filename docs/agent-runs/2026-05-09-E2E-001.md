## 覆盖链路

- 后台确认仓库、渠道、包装和增值服务 demo 配置。
- 用户登录并读取专属仓库地址。
- 用户提交包裹预报，后台扫描入库。
- 用户申请打包，后台审核、设置费用、充值。
- 用户余额支付，后台发货并添加轨迹。
- 用户查询轨迹并确认收货到 `SIGNED`。
- 用户提交手工代购，后台审核、采购、到货并转 `Parcel.IN_STOCK`。
- 代购转入包裹后继续申请打包创建运单。
