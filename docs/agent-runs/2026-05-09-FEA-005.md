# 2026-05-09 FEA-005

## 任务

后台代购商品和订单处理页面。

## 交付

- 新增后台商品管理页 `/products`，覆盖商品、SKU、分类列表、新增、编辑和停用。
- 新增后台代购处理页 `/purchases`，覆盖状态 Tab、列表搜索、详情抽屉、商品明细、采购任务和处理时间线。
- 后台代购支持审核、采购、标记到货、标记异常、取消和到货转 `Parcel.IN_STOCK`。
- 为页面真实能力补齐最小后台商品 CRUD API，以及 `purchases.view` 下的仓库选项 API，避免采购角色依赖完整仓库配置权限。
- 补充后端测试覆盖后台商品管理、代购异常/取消和采购仓库选项权限。
