# 2026-05-09 FEM-002

## 任务

移动 H5 商品、购物车、手工代购入口。

## 交付

- 将移动端 `/home` 从占位页替换为代购商品首页，覆盖商品卡片、商品详情、SKU 选择、数量和加入购物车。
- 将移动端 `/category` 从占位页替换为商品分类入口，复用商品分类筛选和详情查看。
- 将移动端 `/cart` 从占位页替换为真实购物车和确认订单页，覆盖勾选、数量调整、删除、服务费、提交订单和余额支付。
- 将移动端 `/me` 从占位页替换为会员工作台，提供代购订单、手工代购、商品首页、购物车、包裹和运单入口。
- 新增 `/me/purchases` 代购订单页，覆盖状态筛选、订单列表、订单详情、商品明细、采购信息、余额支付和已转包裹查看入口。
- 新增 `/me/purchases/manual` 手工代购页，覆盖商品名称、链接、单价、数量、备注、服务费、提交订单和余额支付。
- 新增移动端代购 API/type 层，复用既有 `products`、`cart-items`、`purchase-orders`、`wallet` 接口，不新增后端接口和依赖。

## 边界

- 不做复杂营销装修，不接真实电商自动采购。
- 移动端复用后端已有 SKU、购物车、代购订单和钱包支付能力，不新增独立移动端业务状态机。
- 支付验收使用本地钱包余额，不包含真实支付网关、支付回调或退款。
- PostgreSQL/MySQL/Redis/Docker 仍不在本轮真实验证范围内。
