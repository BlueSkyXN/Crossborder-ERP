# 2026-05-09 FEU-002

## 任务

用户 Web 手工代购、商品、购物车、确认订单。

## 交付

- 新增用户端代购中心 `/products`、`/cart`、`/purchases`，覆盖商品列表、商品详情、SKU 选择、购物车和确认订单。
- 新增手工代购表单，支持多商品行、商品链接、单价、数量、备注和服务费。
- 新增代购订单列表和详情，覆盖状态筛选、搜索、分页、商品明细、采购进度和转包裹提示。
- 余额支付入口复用现有钱包接口，支付弹窗展示应付金额、账户余额和最近钱包流水。
- 控制台接入商品代购入口，并把待付款代购金额计入待支付金额统计。
- 修正购物车选择语义：初次加载默认全选，但用户手动取消后允许 0 件选择。
