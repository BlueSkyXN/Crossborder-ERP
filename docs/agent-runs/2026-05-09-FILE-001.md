# 2026-05-09 FILE-001

## 任务

补齐本地文件上传基础，支撑包裹图片、汇款凭证、客服附件、商品/内容图片等后续生产化功能。当前仍只真实验证 SQLite 和本地 media，不接对象存储。

## 交付

- 新增 `backend/apps/files`，包含文件元数据模型、SQLite migration、上传/列表/详情/删除/鉴权下载 API。
- 文件上传按业务用途限制大小、扩展名和 MIME；API 返回 `file_id` 与下载 endpoint，不返回本地 `storage_key`。
- 会员只能访问自己的文件；后台可访问后台鉴权范围内文件；包裹图片被业务引用后，对应包裹所属会员可下载。
- 包裹入库图片从任意字符串升级为真实 `PARCEL_PHOTO` file id 校验，并在 Admin Web 入库/扫码入库表单接入上传按钮。
- User Web/Mobile H5 包裹详情显示文件名，不再只展示裸 file id。
- 执行状态推进到 `FIN-001`。
