## 任务

补齐三端浏览器级 E2E 基础，在不污染本机环境、不下载浏览器、不引入随机依赖的前提下，验证 Admin Web、User Web、Mobile H5 的登录和关键页面可用性。

## 交付

- 新增 `npm run e2e:browser`。
- 新增 `scripts/e2e/browser-smoke.sh`，负责临时 SQLite、media、Chrome profile、测试服务启动和清理。
- 新增 `scripts/e2e/browser-smoke.mjs`，使用 system Chrome + Chrome DevTools Protocol 执行三端 smoke。
- CI 新增 `Browser Smoke` job。
- Admin Web/User Web 增加空 favicon，避免浏览器 smoke 因 `/favicon.ico` 404 误判。
- README、部署说明、实施决策、交付审计、差距地图、backlog、known issues、任务图和当前状态同步到 `SHIP-BATCH-001`。

- 不安装全局依赖。
