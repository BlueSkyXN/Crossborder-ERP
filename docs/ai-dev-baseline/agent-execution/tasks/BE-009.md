# BE-009 最小代购和到货转 Parcel

phase: `P4_PURCHASE`  
depends_on: `BE-003`, `BE-005`, `BE-007`  
next: `FEA-005`

## 目标

实现手工代购、自营商品简版、采购处理和到货转 Parcel。

## 必读

- `../../02-domain-model-state-machines.md`
- `../../04-api-database-contract.md`
- `../../14-module-implementation-spec.md`

## 必须做

- Django apps：`products`、`purchases`
- Models：`ProductCategory`、`Product`、`ProductSku`、`CartItem`、`PurchaseOrder`、`PurchaseOrderItem`、`ProcurementTask`
- APIs：
  - `GET /api/v1/products`
  - `GET /api/v1/products/{id}`
  - `POST/GET/PATCH/DELETE /api/v1/cart-items`
  - `POST /api/v1/purchase-orders`
  - `POST /api/v1/purchase-orders/manual`
  - `GET /api/v1/purchase-orders`
  - `POST /api/v1/purchase-orders/{id}/pay`
  - `GET /api/v1/admin/purchase-orders`
  - `POST /api/v1/admin/purchase-orders/{id}/review`
  - `POST /api/v1/admin/purchase-orders/{id}/procure`
  - `POST /api/v1/admin/purchase-orders/{id}/mark-arrived`
  - `POST /api/v1/admin/purchase-orders/{id}/convert-to-parcel`
- 到货转 Parcel 后可继续走集运主链路。

## 不要做

- 不接淘宝/1688/拼多多真实接口。
- 不做自动采购和外链解析。

## 验收

```bash
cd backend
pytest apps/products apps/purchases
```

业务验收：

- 用户提交手工代购。
- 后台审核、采购、到货。
- 到货转 `Parcel IN_STOCK`。
