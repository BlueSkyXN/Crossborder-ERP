next: `FEU-002`

## 目标

实现后台代购订单列表、详情、审核、采购、到货和转 Parcel。

## 必读

- `../../15-frontend-style-from-screenshots-and-repro.md`
- `../../02-domain-model-state-machines.md`
- `../../04-api-database-contract.md`

## 必须做

- 商品分类、商品、SKU 简版管理。
- 代购订单列表和状态 Tab。
- 代购订单详情。
- 审核、采购、到货、异常、取消。
- 到货转 Parcel 操作。

## 不要做

- 不做自动采购。
- 不接外部电商接口。

## 验收

```bash
cd admin-web
npm run lint
npm run build
```

联动验收：

- 后台可把代购到货转成 Parcel。
