# FEM-002 移动 H5 商品、购物车、手工代购入口

phase: `P4_PURCHASE`  
depends_on: `BE-009`, `FEM-001A`  
next: `E2E-001`

## 目标

实现移动端最小商品和代购入口，满足 P0 代购链路在手机端可用。

## 必读

- `../../13-integrated-product-spec.md`
- `../../15-frontend-style-from-screenshots-and-repro.md`
- `../../04-api-database-contract.md`

## 必须做

- 首页商品卡片。
- 分类入口。
- 商品详情。
- 购物车。
- 确认订单。
- 我的页代购订单入口。
- 手工代购入口。

## 不要做

- 不做复杂营销装修。
- 不做自动采购。

## 验收

```bash
cd mobile-h5
npm run lint
npm run build
```

移动验收：

- 375px 宽移动视口下商品、购物车、确认订单流程可操作。
