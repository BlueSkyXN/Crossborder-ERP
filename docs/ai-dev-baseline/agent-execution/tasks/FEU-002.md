# FEU-002 用户 Web 手工代购、商品、购物车、确认订单

phase: `P4_PURCHASE`  
depends_on: `BE-009`, `FEU-001A`  
next: `FEM-002`

## 目标

实现用户 Web 最小代购链路和自营商品基础购物流程。

## 必读

- `../../13-integrated-product-spec.md`
- `../../15-frontend-style-from-screenshots-and-repro.md`
- `../../04-api-database-contract.md`

## 必须做

- 商品列表。
- 商品详情。
- 购物车。
- 确认订单。
- 手工代购表单。
- 代购订单列表和详情。
- 支付入口复用钱包支付。

## 不要做

- 不做商品链接自动解析。
- 不做复杂 SKU 抓取。

## 验收

```bash
cd user-web
npm run lint
npm run build
```

业务验收：

- 用户能提交手工代购。
- 后台处理到货后，用户能看到生成的 Parcel。
