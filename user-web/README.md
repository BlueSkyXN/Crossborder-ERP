# User Web

用户 Web 将使用 React、Vite、TypeScript、React Router、TanStack Query、Axios、Zustand、CSS Modules 和共享 UI tokens。

当前已完成 `FEU-001A` 基础入口：

- 会员登录和本地登录态。
- 会员控制台、会员资料、余额占位和状态入口占位。
- 专属仓库地址卡片和一键复制。
- loading、empty、error 基础状态。

本地开发命令：

```bash
pnpm --filter @crossborder-erp/user-web dev
pnpm --filter @crossborder-erp/user-web lint
pnpm --filter @crossborder-erp/user-web build
```
